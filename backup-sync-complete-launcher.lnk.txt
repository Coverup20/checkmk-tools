Powershell Launcher Shortcut

Target:
powershell.exe -NoExit -ExecutionPolicy Bypass -File "C:\Users\Marzio\Desktop\CheckMK\Script\backup-sync-complete.ps1"

Start in:
C:\Users\Marzio\Desktop\CheckMK\Script

Icon:
powershell.exe

Istruzioni:
1. Clicca col destro su desktop → Nuovo → Collegamento
2. Incolla il Target sopra come percorso
3. Imposta la cartella di avvio su "Start in"
4. Cambia icona se vuoi
5. Testa il collegamento manualmente
6. Se funziona, puoi farlo lanciare dal task tramite batch o direttamente
